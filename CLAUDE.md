# BMK CRM Platform

## Project Overview
A custom CRM and AI-powered client onboarding platform for BMK Financial Services (Brad Lonergan, Newcastle NSW). Replaces manual financial planning workflows with an automated pipeline.

## Stack
- Next.js 14, TypeScript, Tailwind CSS, shadcn/ui
- App Router
- Deployed on Vercel

## Rules
- Never use inline styles
- Never create new branches — always commit to main
- Use shadcn/ui components, never build from scratch
- Keep components small and focused

## Current Build Phase
Phase 1 — App shell and navigation only. No integrations yet.
